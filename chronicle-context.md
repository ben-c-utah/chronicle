# Chronicle Project Context

Chronicle is a historical companion platform designed primarily for tabletop wargamers.

It provides an interactive historical reference system built around:

- a map of historical events
- a chronological timeline
- encyclopedia-style records
- cultural and technological overviews
- campaign narratives
- hobby-oriented journal posts

Chronicle is currently focused on the Viking Age but is intentionally designed to support other eras.

The project follows a static-first architecture.

Content is stored as MDX files and rendered through a Next.js application.

The documentation in this repository defines the full architecture of the project.

Please read the documentation files before proposing implementation steps.
