
# Chronicle Journal System
Version 1.0

The Chronicle Journal is an editorial section of the Chronicle project.

It functions similarly to a blog and allows sharing hobby content and project updates.

---

# Purpose

The Journal allows Chronicle creators to:

- share miniature painting
- document terrain builds
- publish battle reports
- record campaign stories
- discuss historical research

# Update: Video Support

Journal posts commonly include video content.

Examples:

- miniature painting tutorials
- terrain construction walkthroughs
- battle report videos
- historical explanation videos

Video should normally be embedded from:

- YouTube
- Vimeo

Videos are stored using the `videoGallery` structure.
