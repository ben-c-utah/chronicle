
# Chronicle Visual Design System
Version 1.0

Chronicle is designed to feel like a **historical manuscript adapted into a modern digital tool**.
The aesthetic draws inspiration from medieval chronicles, illuminated manuscripts, and monastic scriptoria
while maintaining modern usability and readability.

---

# Core Design Philosophy

Historical Atmosphere: 25%
Modern Usability: 75%

If a stylistic choice conflicts with readability or usability, usability takes priority.

Chronicle should feel calm, scholarly, and immersive.

# Layout Structure

Chronicle primarily uses a two‑panel layout.

Left: Map / Exploration
Right: Chronicle Panel

Map panel:
- restrained styling
- minimal ornamentation
- muted map colors

Chronicle panel:
- parchment background
- elegant serif typography
- decorative manuscript borders

---

# Decorative Elements

## Illuminated Initials
Large decorative first letter for chronicle entries.

## Decorative Dividers

# Iconography

Suggested motifs:
- raven
- longship
- quill
- wax seal
- sword
- church
- crown

Icons should resemble engraved or manuscript illustrations.

---

# Animation Style

Animations should be subtle and dignified.

Examples:
- Chronicle panel slides like a turning page
- Map pans smoothly
- Marker highlights glow gently

---

# Accessibility

- strong text contrast
- minimum body text size 16px
- decorative fonts never used for body text
- generous line spacing
